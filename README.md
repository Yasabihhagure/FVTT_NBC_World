# FVTT用 信長の黒い城 (Nobunaga's Black Castle)

TRPG「信長の黒い城」のFoundry VTT用システムモジュールです。

## 概要

このシステムはFoundry VTT上で「信長の黒い城」をプレイするための各種機能を実装します。
MORK BORGの派生システムとしての特徴的な戦闘ルールや、独自の判定ロジックをサポートします。

## 機能 (予定含む)

*   **キャラクター管理**: 簡易キャラクターおよび上級キャラクターのデータ管理
*   **NPC・敵キャラクター**: プリセットからの敵配置、攻撃サイクルの管理
*   **グループ管理**: パーティーおよび敵グループの編成と士気チェック (Morale Check)
    *   **部隊士気チェックの利用条件**:
        1.  アクターが「フォルダ」に整理されていること。（フォルダ単位でグループ識別を行います）
        2.  ワールドに「シーン」が作成され、アクティブになっていること。
        3.  グループメンバーのトークンがシーン上に配置されていること。
*   **戦闘支援**: 推奨攻撃手段の提示、ダメージ適用

## インストール

現在開発中です。
`system.json` のリンクを使用してインストールを行ってください。
